📱📨 WebChat 

**Single Page Application (SPA)**  sviluppata per simulare una webchat dinamica client side rendering grazie alla gestione delle pagine e delle single-chat tramite Tailwind4 e Javascript. 

### 🛠️Tech Stack
- HTML5: semantica seo friendly e accessibile.
- TAILWIND4: Personalizzazione di elementi, componenti responsive e gestione navigazione tramite classi.
- JAVASCRIPT: Gestione Navigazione con ausilio di Tailwind, ascolto degli eventi, creazione dinamica di elementi DOM.

### 🥁Funzionalità Implementate

**Layout Responsive Avanzato**:
- Approccio *Mobile-First*.
- Navigazione adattiva: Sidebar fissa su Desktop, barra inferiore e overlay a schermo intero su Mobile.
  
**Sistema di Navigazione SPA**:
- Transizioni fluide tra le viste (Lista contatti, Chat Singola, Chat di Gruppo) gestite tramite toggling di classi Tailwind (`hidden`). 

**Modulo Chat**:
- **Chat di Gruppo**: Vista statica per simulazione conversazione multi-utente.
- **Chat 1-to-1 (Interattiva)**: Conversazione simulata con "Laura Laurense". Supporto invio messaggi tramite tasto `Invio` o bottone, con risposta automatica differita.
  
**Feature Social**:
- **Stories**: UI dedicata con simulazione di stato "in arrivo".
- **Ricerca**: Input di ricerca presente nell'UI (pronto per futura implementazione logica).

### Obiettivo: 
**Creare una Web-Chat Realistica con Tailwind4**
- Eliminazione del ricaricamento della pagina (SPA Architecture)
- Consolidamento della UX attraverso l’approccio “Mobile First”
- Simulazione di interattività realistica (Chat 1 a 1) senza Backend



### 🛫 Disponibile in Locale

- Clona il repo: ` git clone https://github.com/greenexplorerdev/Web-Chat-Application.git `
